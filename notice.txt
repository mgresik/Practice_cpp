Память для статических объектов выделяется автоматически, на этапе компиляции. Взаимодействие с ними происходит исключительно с помощью имён.
Память для динамических объектов указывается вручную, работа с ними - через указатели! Памятью управлять с помощью "new" и "delete".

Важное про break return continue: бряка кидает за цикл, а ретурн выбрасывает в calller, вообще из функции. Контюна позволяет пролететь код и переместит в самый конец блока.
